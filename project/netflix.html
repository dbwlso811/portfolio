Project Goal

기존의 레이아웃을
심플하게 재구성.
웹접근성과 가독성 확보.
디자인된 레이아웃을 능동적이고
다양한 움직임을 줌.

UI / UX Design

- 텍스트를 최대한 심플하게 나타내어
깔 끔하게 만듬.

- 모든 사용자 정보를 쉽게 인지하고
사용할 수 있도록, 크고 넓은 레이아웃 을 가짐.

- 가로스크롤로
더 심플하고 깔끔하게 디자인함.

Publishing

- 스크린 사이즈에 따라 레이아웃이 변하는
반응형 웹 구현.

- 사용자의 접근성을 높이기위해 텍스트를
이미지 대신 마크업 하여 웹 표준성을 준수.

- 동적인 요소를 위해 jQuery 플러그인,
Javascript를 사용.

- 메인에
swiper 사용.
